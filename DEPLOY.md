# Публикация сайта в открытый доступ

Сайт уже готов для Node-хостинга: `server.js` отдаёт страницу и API `/api/quotes`, а порт берётся из переменной `PORT`.

## Вариант 1: Render

1. Создай репозиторий на GitHub и загрузи туда файлы проекта.
2. Открой Render и выбери `New` -> `Web Service`.
3. Подключи GitHub-репозиторий.
4. Render сам увидит `render.yaml`.
5. После деплоя появится публичная ссылка вида:
   `https://investment-portfolio-dashboard.onrender.com`

## Вариант 2: Railway

1. Создай проект в Railway.
2. Подключи GitHub-репозиторий.
3. Start command: `npm start`.
4. Railway сам выдаст публичный домен.

## Важно

После деплоя сайт будет работать без включённого компьютера и без локальной сети. Котировки будут обновляться через сервер хостинга.
